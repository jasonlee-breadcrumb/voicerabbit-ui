Voice Rabbit — Recording Pill

Developer reference & animation spec

Push to talk

No stop button. User holds hotkey → pill appears. Release → pill disappears. Waveform reacts to voice in real time.

Hands-free mode

User presses hotkey once → pill stays on screen with stop button. Press stop or hotkey again to end.

Dev notes

  • Waveform uses Web Audio API + AnalyserNode — 6 frequency bands mapped to bar heights.
  • When mic is active: bars driven by getByteFrequencyData() at 60fps via requestAnimationFrame.
  • When silent / no speech: bars smoothly return to min height (~4px) — looks "flat".
  • Fallback: if mic permission denied, bars run a CSS looping animation.
  • In the Electron Mac app, mic access is always available — no permission prompt needed after initial setup.
  • Pill entrance: pillIn keyframe — 250ms spring ease, slight translateY + scale.
  • Pill exit: fade + scale down, 150ms. Trigger on hotkey release (push) or stop button tap (hands-free).
